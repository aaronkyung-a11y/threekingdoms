## v1.2 (2026-07-12)
PWA(설치형 웹앱)化 — 홈 화면 설치·전체화면·오프라인 실행.

### PWA 기능
- **manifest.json**: 앱 이름·아이콘·standalone(전체화면)·세로 고정
- **서비스워커(sw.js)**: 앱 파일 오프라인 캐시(캐시 우선 전략) → 인터넷 없이 실행
- **앱 아이콘**: 192·512px + maskable(금색 '群' 한자)
- **설치 버튼**: Android/Chrome은 원터치 설치, iOS는 수동 설치 안내(홈 화면에 추가)
- **iOS/노치 대응**: safe-area-inset 패딩, apple-touch-icon, 상태바 스타일
- 100dvh로 모바일 주소창 변화에도 꽉 찬 화면

### 배포 파일 (함께 호스팅 필요)
- index.html, manifest.json, sw.js, icon-192.png, icon-512.png, icon-maskable-512.png
- 정적 호스팅(GitHub Pages, Netlify, Vercel 등) HTTPS에 올리면 설치 가능

### 검증

## v1.0 (2026-07-12)
시나리오 선택 + 지역 지형 반영 + 배경음악 + 극적 특수효과.

### 1. 다양한 시나리오
- **3개 시나리오** 선택 가능:
  - 190 반동탁 연합 — 표준 시나리오, 동탁 최강·연합 이벤트
  - 194 군웅할거 — 동탁 사후, 조조 연주 확장·유비 서주·손책 강동, 절대강자 없음
  - 200 관도대전 — 조조 vs 원소 양대 세력 격돌, 다수 군소세력 멸망
- 시나리오별 세력 배치·무장 소속/생사·플레이 가능 세력·시작 연도가 다름
- 시나리오 → 세력 선택 2단계 흐름, 세력별 도시 수·난이도 자동 표시

### 2. 전투 맵 실제 지형 반영
- 방어 도시의 지리에 따라 전투 지형이 달라짐:
  - 형주·강동(양양·강릉·강하 등) → 강 많음
  - 익주·한중·서량(성도·재동·한중 등) → 산악
  - 중원(허창·진류 등) → 평야
- 30여개 도시에 고유 지형 프로파일(강/산/숲 밀도) + 지명 라벨(촉 분지, 장강 강변 등)
- 전투 제목에 지형명 표시

### 3. 배경음악 (절차적 생성)
- 평시(field): 오리엔탈 5음계 차분한 멜로디
- 전투(battle): 빠른 템포 + 북소리로 박진감(삼국지풍)
- 전투 진입/종료 시 자동 전환, 🔊 버튼으로 on/off

### 4. 극적 특수효과
- 전체화면 임팩트 플래시(강타·궤멸 시)
- 파티클 폭발(병사 흩어짐, 궤멸 시 대량)
- 참격 슬래시 이펙트
- 일기토 집중선(만화적 연출)
- 기존 화면흔들림·데미지팝업·타격플래시와 결합

### 검증
- 3개 시나리오 배치·무장 이동/생사·지형 프로파일·BGM(오디오 노드 생성)·극적효과 전부 통과 (JS 에러 0)

## v0.9 (2026-07-12)
상단 UI 공간 압축 — 지도 영역 확대.

### 문제
- 상단바가 세로로 여러 줄 차지(제목·자원이 줄바꿈, '반동탁연합' 배지가 세로로 늘어남)
- 화면 상단이 30~35%를 먹어 지도가 좁았음

### 개선
- 상단바를 **2줄 고정**으로 재설계 (1줄: 제목·날짜·연합배지·저장/불러오기 / 2줄: 세력·자원)
- 자원 표기 압축(금/군량/병/성) + nowrap(줄바꿈 방지, 넘치면 가로 스크롤)
- 저장/불러오기 버튼을 아이콘(💾/📁)으로 축소
- 연합 배지 '反董卓聯合' 한 줄 고정(세로로 안 늘어남)
- 날짜 표기 간결화(서기 생략)
- 메시지 스트립 66→48px
- **결과: 상단 영역 화면의 ~35% → 12%, 지도가 88% 차지**

### 검증
- 상단바 62px·메시지 48px로 측정, 자원줄 wrap 없음
- 저장/패널/턴진행/연합배지 정상 (JS 에러 0)

### 다음 후보
- 무장 이동·등용, 계략, 전투 애니메이션, 배경음악
